🥫 Pocket Pantry

Pocket Pantry is a smart kitchen companion app designed to help users track pantry items, reduce food waste, and stay organized with their groceries. Built with a clean, modern UI, the app makes it easy to know what you have, what you need, and what’s about to expire—all in one place.

✨ Features

📦 Inventory Tracking
Keep a real-time list of all pantry items
⏰ Expiration Monitoring
Quickly see which items are expiring soon
🛒 Smart Grocery Planning
Avoid duplicate purchases and plan smarter shopping trips
🧠 Simple & Intuitive UI
Designed for ease of use with a clean, responsive layout
📱 Cross-Device Friendly
Optimized for a smooth experience across devices
